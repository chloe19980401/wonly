GH

GitHub Pages

把当前文件夹上传到 GitHub 仓库，启用 Pages 后即可公开访问。

  1. 创建仓库，例如 wonly-social-dashboard
  2. 上传 index.html、styles.css、app.js 和 assets
  3. 在仓库 Settings → Pages 选择主分支发布
DNS

自定义域名

我已准备 CNAME 文件，你把里面的域名替换成自己的即可。

  1. 在域名 DNS 添加 CNAME 到 GitHub Pages 地址
  2. 在 GitHub Pages 填入同一个域名
  3. 等待 HTTPS 证书自动签发
